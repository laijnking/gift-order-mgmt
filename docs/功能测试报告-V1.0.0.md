## 一、测试概述

### 1.1 测试目的
对礼品订单管理系统（OBS）进行全面功能测试，验证系统核心业务流程、数据管理、API接口等功能的正确性和稳定性。

### 1.2 测试范围
- 档案管理模块（客户、供应商、仓库、商品、SKU映射）
- 订单管理模块（导入、匹配、导出）
- 发货与回单模块
- 历史数据管理（历史成本库、价格历史、库存版本）
- 系统设置（用户、角色、权限、模板、预警规则）

### 1.3 测试数据来源
- `assets/订单/` - 订单Excel文件和文本订单数据
- `assets/供应商发单/` - 供应商发单模板和数据
- `assets/历史发货成本资料含运费20260330.xlsx` - 历史成本数据
- `assets/商品编码-订单表以此表商品编码为准.xlsx` - 商品编码数据
- `assets/库存汇总.xlsx` - 库存数据
