## 重要决策记录

### 决策 1：数据库选型

**问题**: 选择什么数据库？

**方案**: Supabase (PostgreSQL)

**理由**:
- 关系型数据适合订单场景
- 内置 RLS 安全策略
- 实时订阅能力
- 管理后台开箱即用

**日期**: 2024-01

### 决策 2：库存预警阈值

**问题**: 尾货预警阈值设为多少？

**方案**: 库存 ≤ 2 台

**理由**:
- 礼品代发通常批量较小
- 2 台库存风险较高
- 兼顾敏感度和噪音

**日期**: 2024-01

### 决策 3：订单号生成

**问题**: 如何生成系统订单号？

**方案**: SYS-YYYYMMDD-XXXX

**理由**:
- 格式清晰易读
- 日期分区方便
- 序号保证唯一

**日期**: 2024-01

## 问题与解决

### 问题 1：Excel 表头识别

**问题描述**: 不同客户 Excel 表头格式不一致

**解决方案**:
- 模糊匹配表头名称
- 支持多个候选列名
- 记录匹配规则

### 问题 2：库存版本同步

**问题描述**: 库存变更需要记录历史

**解决方案**:
- 创建 stock_versions 表
- 库存更新时自动创建版本记录
- 支持按商品、时间筛选

### 问题 3：供应商匹配效率

**问题描述**: 匹配算法需要考虑多个因素

**解决方案**:
- 仓库类型优先
- 尾货预警提示
- 价格综合评估
- 快递限制过滤
