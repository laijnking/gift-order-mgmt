## 一、SKU映射说明

SKU映射用于建立**客户商品/供应商商品**与**系统商品档案**之间的对应关系，实现商品信息的自动匹配。

### 1.1 为什么需要SKU映射？

```
客户订单商品：「苏泊尔ZMD安心系列绞肉机」
           ↓
      无法直接匹配系统商品
           ↓
    需要SKU映射建立对应关系
           ↓
系统商品：「苏泊尔ZMD安心系列 升级搅菜杆双层四刃精钢刀锋绞肉机」
```

### 1.2 映射类型

| 映射类型 | 说明 | 使用场景 |
|----------|------|----------|
| 客户商品映射 | 客户Excel商品 → 系统商品 | 订单导入时匹配 |
| 供应商商品映射 | 供应商Excel商品 → 系统商品 | 库存导入时匹配 |

## 四、填写注意事项

### 4.1 映射优先级
- 当同一客户/供应商的同一商品可能有多个映射时使用
- 数字越小，优先级越高
- 通常填写 `1`

### 4.2 客户商品名称规范
- 填写客户Excel中**实际使用的商品名称**
- 注意全角半角、空格等细微差异
- 例如：`苏泊尔绞肉机` ≠ `苏泊尔 绞肉机`

### 4.3 系统商品信息
- 必须从系统商品档案中获取
- 可先导出系统商品档案，再进行映射

### 4.4 映射规则

```
┌─────────────────────────────────────────────────────────┐
│ 匹配优先级（从高到低）：                                 │
│                                                          │
│ 1. 规格型号精确匹配                                      │
│ 2. 商品编码精确匹配                                      │
│ 3. 商品名称模糊匹配                                      │
│ 4. SKU映射表匹配                                        │
└─────────────────────────────────────────────────────────┘
```

## 五、导入流程

### 5.1 客户商品映射导入
```
1. 整理客户商品映射Excel
2. 在「档案管理 > SKU映射」页面导入
3. 系统自动校验映射关系
4. 确认导入完成
```

### 5.2 供应商商品映射导入
```
1. 整理供应商商品映射Excel
2. 在「档案管理 > SKU映射」页面导入
3. 系统自动校验映射关系
4. 确认导入完成
```
