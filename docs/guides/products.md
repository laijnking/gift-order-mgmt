# 商品管理 - 操作指引

更新时间：2026-05-12

## 1. 功能说明

商品管理模块负责管理商品档案，支持商品的增删改查、批量导入导出，以及品牌分类和尺寸重量管理。

---

## 2. 操作流程

### 2.1 商品查询

#### 基本查询
1. 进入「商品管理」页面
2. 系统默认显示所有商品记录
3. 可按以下条件筛选：
   - 商品编码/名称/条码（搜索框）
   - 品牌（多选下拉）
   - 生命周期状态（正常/停产/清仓）

#### 查看详情
1. 点击商品记录可查看详情
2. 显示信息包括：
   - 商品基本信息（编码、名称、条码、品牌、分类）
   - 规格型号和单位
   - 成本价和零售价
   - 尺寸和重量信息（用于抛货判断和运费计算）
   - 生命周期状态

### 2.2 新建商品

1. 点击「新建商品」按钮
2. 填写商品信息：
   - **商品编码**：必填，系统唯一标识
   - **条码**：可选，商品条码
   - **商品名称**：必填，商品名称
   - **品牌**：可选，商品品牌
   - **分类**：可选，商品分类
   - **规格型号**：可选，规格说明
   - **单位**：默认"台"
   - **成本价/零售价**：可选
   - **尺寸信息**：长、宽、高（cm）
   - **重量**：kg
   - **抛货系数**：默认 6000，用于体积重量计算
3. 点击「保存」

### 2.3 编辑商品

1. 点击商品行的「编辑」按钮
2. 修改商品信息
3. 点击「保存」

### 2.4 删除商品

1. 点击商品行的「删除」按钮
2. 确认删除操作

## 3. Excel 导入

### 3.1 下载导入模板

1. 点击「模板下载」按钮
2. 系统下载 `商品管理_导入模板.xlsx`

### 3.2 导入模板字段

| Excel列名 | 系统字段 | 说明 |
|-----------|----------|------|
| 商品编码 | code | 必填，唯一标识 |
| 条码 | barcode | 可选 |
| 商品名称 | name | 必填 |
| 品牌 | brand | 可选 |
| 分类 | category | 可选 |
| 规格型号 | spec | 可选 |
| 单位 | unit | 可选，默认"台" |
| 成本价 | costPrice | 可选 |

| 宽度(cm) | widthCm | 可选 |
| 高度(cm) | heightCm | 可选 |
| 重量(kg) | weightKg | 可选 |

| 备注 | remark | 可选 |

### 3.4 导入流程

1. 点击「导入」按钮
2. 选择 Excel 文件（.xlsx/.xls）
3. 系统解析并显示预览
4. 点击「确认导入」
5. 系统分批导入（每批100条）
6. 显示导入结果统计

## 5. 字段说明

### 5.1 尺寸和重量字段

| 字段 | 说明 | 用途 |
|------|------|------|
| lengthCm | 长度(cm) | 体积计算、运费预估 |
| widthCm | 宽度(cm) | 体积计算、运费预估 |
| heightCm | 高度(cm) | 体积计算、运费预估 |
| weightKg | 重量(kg) | 实际重量、运费计算 |
| volumeFactor | 抛货系数 | 默认6000，体积重量 = 长×宽×高÷抛货系数 |

### 5.2 体积重量计算

```
体积重量 = 长(cm) × 宽(cm) × 高(cm) ÷ 抛货系数
```

例如：30cm × 25cm × 20cm ÷ 6000 = 2.5kg（体积重量）

### 5.3 生命周期状态

| 状态值 | 显示名称 | 说明 |
|--------|----------|------|
| `active` | 正常 | 商品在售 |
| `discontinued` | 停产 | 商品已停产 |
| `clearance` | 清仓 | 商品清仓处理中 |

## 7. 常见问题

### Q: 导入时提示"商品编码已存在"？
A: Excel 中商品编码与系统中已有商品重复。检查并修改 Excel 中的商品编码。

### Q: 尺寸信息有什么用途？
A: 用于计算体积重量，在订单派发时预估快递费用。抛货商品（轻但体积大）按体积重量计费。

### Q: 如何设置抛货系数？
A: 抛货系数默认6000，可在新建/编辑商品时修改。数值越小，体积重量越大。
