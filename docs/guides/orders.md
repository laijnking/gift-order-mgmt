# 订单管理 - 操作指引

更新时间：2026-04-29

## 1. 功能说明

订单中心是系统的核心模块，负责管理所有订单的生命周期，支持订单导入、状态流转、筛选搜索、批量派发和详情查看。

---

## 2. 操作流程

### 2.1 查看订单列表

1. 进入「订单管理」页面，默认显示所有订单
2. 页面顶部显示统计信息：
   - 总订单数
   - 未归档订单数（状态非「已完成」或「已取消」）
   - 待派发订单数

### 2.2 筛选订单

**基础筛选**：
1. 点击状态筛选下拉框
2. 选择要筛选的状态（可多选）
3. 支持按「待派发」「回单阶段」「已归档」等快速筛选

**高级筛选**：
1. 点击「高级筛选」展开筛选面板
2. 支持按以下条件筛选：
   - 客户
   - 发货方
   - 数量范围
   - 导入批次
   - 时间范围
   - 按导入批次回看

### 2.3 订单操作

#### 查看详情
1. 点击订单行的「眼睛」图标
2. 在弹出的详情对话框中查看：
   - 订单基本信息
   - 商品明细（含派发时的发货方商品编码快照）
   - 物流信息
   - 关联的发货方信息

#### 编辑订单
1. 点击订单行的「编辑」图标
2. 根据订单状态，可编辑的字段有所不同：
   - `pending`：可编辑所有信息
   - `assigned`：可编辑发货方、状态、物流信息、收货信息
   - `partial_returned`：可编辑发货方、状态、物流信息、收货信息
   - `returned`/`feedbacked`：仅可编辑收货信息
   - `cancelled`：可编辑所有信息

#### 删除订单
1. 点击订单行的「删除」图标
2. 确认删除操作

### 2.4 批量派发

1. 勾选要派发的订单（状态为「待派发」）
2. 点击底部的「派发」按钮
3. 在派发对话框中：
   - 点击「智能匹配」自动推荐发货方
   - 或手动为每条订单选择发货方
4. 确认派发

### 2.5 批量操作

**批量回单**：
1. 勾选要回单的订单
2. 点击底部的「回单」按钮
3. 输入快递单号（支持批量导入，每行一个）
4. 确认回单

**导出发货通知**：
1. 勾选已派发的订单（状态为「已派发」）
2. 点击底部的「发货通知」按钮
3. 系统跳转到发货导出页面

**客户反馈**：
1. 勾选已回单的订单（状态为「已回单」）
2. 点击底部的「反馈」按钮
3. 系统自动复制反馈内容到剪贴板
4. 订单状态更新为「已反馈」

**导出金蝶**：
1. 勾选已反馈的订单（状态为「已反馈」）
2. 点击底部的「导出金蝶」按钮
3. 系统导出 Excel 文件并更新订单状态为「已完成」

## 3. 订单状态说明

| 状态值 | 显示名称 | 说明 | 可流转状态 |
|--------|----------|------|------------|
| `pending` | 待派发 | 订单已导入，等待分配发货方 | assigned, cancelled |
| `assigned` | 已派发 | 已分配发货方（内部派单） | notified, partial_returned, returned, cancelled |
| `notified` | 通知发货 | 已导出发货通知单 | partial_returned, returned, cancelled |
| `partial_returned` | 部分回单 | 部分商品已回单 | returned, completed |
| `returned` | 已回单 | 发货方已回传快递单号 | completed |
| `feedbacked` | 已反馈 | 已向客户反馈物流信息 | - |
| `completed` | 已完成 | 订单完结 | - |
| `cancelled` | 已取消 | 订单取消 | - |

> **注意**：`assigned`（已派发）是内部操作，`notified`（通知发货）是外部通知。状态流转：`pending → assigned → notified → partial_returned/returned → completed`

## 4. 数据流转

### 4.1 订单数据来源

```
订单录入 (order-parse)
    ↓
写入 orders 表
    ↓
订单中心 (orders) ← 显示/编辑
    ↓
派发 → dispatch_records 表（派发快照）
```

### 4.2 订单状态变更

```
pending (待派发)
    ↓ 派发
assigned (已派发)
    ↓ 导出发货通知
notified (通知发货)
    ↓ 回单
returned (已回单)
    ↓ 反馈
feedbacked (已反馈)
    ↓ 导出金蝶
completed (已完成)
```

| 入口 | 目标页面 | 说明 |
|------|----------|------|
| 新增订单 | - | 手动创建订单 |

| 客户反馈 | `/feedback-export` | 导出客户反馈 |

## 6. 注意事项

### 6.1 编辑限制
- `completed` 状态订单无法编辑
- `feedbacked` 状态订单仅可编辑收货信息
- `partial_returned` 状态订单编辑功能受限

### 6.2 删除限制
- 只有 `pending` 和 `cancelled` 状态的订单可以删除
- 已派发或更后状态的订单需要先取消

### 6.3 派发说明
- 同一订单重复派发时会复用既有派发记录，不会重复扣减库存
- 派发时快照写入发货方商品编码，导出时直接从快照读取
