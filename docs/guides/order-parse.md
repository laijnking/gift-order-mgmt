# 订单解析 - 操作指引

更新时间：2026-04-29

## 1. 功能说明

订单解析模块支持通过 Excel 导入订单，自动解析表头、智能匹配商品和发货方、历史映射自动加载，减少人工录入工作量。

---

## 2. 操作流程

### 2.1 基本操作步骤

#### 步骤1：选择客户
1. 在左侧面板选择「关联客户」
2. 选择后，系统自动联动业务员和跟单员（来自客户档案的绑定关系）

#### 步骤2：上传 Excel
1. 点击上传区域或拖拽文件
2. 支持 `.xls` 和 `.xlsx` 格式
3. 上传后选择要解析的 Sheet

#### 步骤3：配置列映射
1. 系统自动检测表头并尝试自动映射
2. 如有历史映射（按表头指纹 SHA256 精确匹配），自动加载并折叠
3. 可手动调整映射关系

#### 步骤4：解析并预览
1. 点击「解析」按钮
2. 系统解析 Excel 并匹配商品
3. 解析完成后在右侧预览订单

#### 步骤5：提交订单
1. 检查预览中的订单信息
2. 修正错误或补充缺失信息
3. 点击「提交订单」

### 2.2 商品匹配说明

系统按以下优先级自动匹配商品：

| 优先级 | 匹配方式 | 说明 |
|--------|----------|------|
| 1 | 规格匹配 | 客户商品规格 精确匹配 系统商品规格 |
| 2 | 编码匹配 | 客户商品编码 精确匹配 系统商品编码 |
| 3 | 条码匹配 | 客户商品条码 精确匹配 系统商品条码 |
| 4 | 名称匹配 | 客户商品名称 模糊匹配 系统商品名称 |

匹配结果：
- **matched**：精确匹配
- **fuzzy**：模糊匹配（需人工确认）
- **unmatched**：未匹配（需手动选择）

### 2.3 发货方匹配规则

匹配条件：
1. 商品在发货方有库存（quantity > 0）
2. 商品与收货地址省份兼容（无快递限制）
3. 按综合成本排序（商品单价 + 预估运费）

优先级：
1. 内部仓库优先（type = 'self'）
2. 成本最优
3. 库存充足（库存 > 订单数量）

### 2.4 尾货预警

当库存 ≤ 2 台时，系统会显示尾货预警提示，防止撞单风险。

## 3. 列映射说明

### 3.1 支持的字段

| Excel列名 | 系统字段 | 说明 |
|-----------|----------|------|
| 商品名称 | product_name | 必填，客户商品名称 |
| 商品编码 | product_code | 选填，客户商品编码 |
| 规格型号 | product_spec | 选填，客户商品规格 |
| 数量 | quantity | 默认 1 |
| 单价 | price | 选填 |
| 收货人 | receiver_name | 必填 |
| 收货电话 | receiver_phone | 必填 |
| 收货地址 | receiver_address | 必填 |
| 客户订单号 | order_no | 选填，不填则自动生成 |
| 单据日期 | bill_date | 选填 |
| 备注 | remark | 选填 |

### 3.2 历史映射自动加载

```
1. 选择客户后，系统根据 Excel 表头计算指纹（SHA256）
2. 查询该客户是否有完全匹配的映射历史（按 header_fingerprint）
3. 若命中，自动加载映射（绿色提示），默认折叠列映射区域
4. 用户手动修改任一列时，自动加载标记取消
5. 保存映射时，自动构建 feedback_export_headers 存入数据库
```

## 4. 数据流转

### 4.1 Excel 解析流程

```
1. 用户上传 Excel 文件
2. 系统读取整张 sheet 数据
3. 自动检测列名映射
4. 按订单号分组，生成订单列表
5. 返回解析结果供用户确认
```

### 4.2 重复导入识别

```
同批重复：同一批导入请求中，相同客户订单号只保留第一条
系统已存在：同客户同订单号在系统中已存在时跳过
结果弹窗展示：创建数量、跳过数量、重复原因、已存在系统单号
```

### 4.3 订单提交后数据流向

```
订单提交
    ↓
POST /api/orders
    ↓
orders 表 ← 订单主数据
order_items 表 ← 商品明细
column_mappings 表 ← 映射配置（首次提交时自动保存）
    ↓
跳转订单中心
```

## 5. 快捷操作

| 操作 | 说明 |
|------|------|
| 智能匹配发货方 | 为所有订单自动匹配最优发货方 |
| 批量选择/取消 | 全选或取消所有订单 |
| 复制订单 | 复制一条订单记录 |
| 新增订单 | 手动添加一条新订单 |
| 映射历史 | 查看和恢复历史映射配置 |

---

## 6. 注意事项

### 6.1 必填字段
- 商品名称
- 收货人
- 收货电话
- 收货地址

### 6.2 发货方选择
- 未选择发货方的订单将标记为「待派发」状态
- 可在订单中心后续手动派发

### 6.3 历史映射
- 映射按表头内容指纹匹配，不同表头结构需重新配置
- 每次提交会自动保存当前映射
