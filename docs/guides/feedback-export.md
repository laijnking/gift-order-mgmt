# 反馈导出 - 操作指引

更新时间：2026-04-29

## 1. 功能说明

反馈导出模块用于导出已回单订单给客户，支持使用客户导入时的原始列名，确保导出格式与客户导入格式一致。

---

## 2. 核心原则

> **客户用什么列名导入，系统就用什么列名导出。**
> 物流信息（快递公司、运单号）由系统追加，不占用客户原始列。

---

## 3. 操作流程

### 3.1 基本操作步骤

#### 步骤1：选择客户
1. 进入「反馈导出」页面
2. 页面显示有待导出订单的客户列表
3. 选择要导出反馈的客户（可多选）

#### 步骤2：确认导出条件
1. 确认要导出的订单状态（默认：已回单 `returned`）
2. 可选择其他状态组合

#### 步骤3：选择模板（可选）
1. 如需自定义导出格式，可选择导出模板
2. 不选择则使用客户历史映射的列名

#### 步骤4：导出
1. 点击「导出反馈」按钮
2. 系统按客户打包生成 Excel 文件
3. 下载 ZIP 压缩包

---

## 4. 客户原始列名还原

### 4.1 还原原理

系统在客户导入订单时，会保存原始列名映射关系到 `feedback_export_headers` 字段：

```json
{
  "货号": "product_code",
  "品名": "product_name",
  "规格": "product_spec",
  "数量": "quantity"
}
```

导出时，系统会：
1. 查询客户的 `feedback_export_headers`
2. 按原始列名构建 Excel 表头
3. 填充对应的系统数据
4. 在最后追加物流列（快递公司、运单号）

### 4.2 列名还原流程

```
客户上传 Excel
    ↓
识别 customerCode（客户编码）
    ↓
计算 headerFingerprint（SHA256）
    ↓
查询历史映射
    ↓
命中？──否──▶ 用户手动映射 → 保存时指定导出列名 → 存入 feedback_export_headers
    ↓ 是
自动加载映射
    ↓
用户点"提交" → 订单提交
    ↓
...
导出客户反馈单
    ↓
查询 feedback_export_headers
    ↓
按原始列名构建 Excel 表头 + 数据
    ↓
追加物流列（快递公司、运单号）在最后
```

---

## 5. 数据流转

### 5.1 反馈导出流程

```
选择客户
    ↓
查询有已回单订单的客户
    ↓
用户选择导出客户
    ↓
查询反馈导出列名映射
    ↓
构建导出数据（原始列名 + 物流信息）
    ↓
生成 Excel 文件
    ↓
压缩为 ZIP 下载
```

### 5.2 关键 API

| API | 方法 | 说明 |
|-----|------|------|
| `/api/customers/feedback-pending` | GET | 返回有待导出订单的客户列表 |
| `/api/export-feedback/returned-batch` | POST | 导出已回单订单（仅 `returned` 状态） |
| `/api/export-feedback/batch` | POST | 通用导出（可指定状态） |

---

## 6. 导出内容

### 6.1 表头格式

```
[客户原始列1]  [客户原始列2]  ...  [客户原始列N]  [快递公司]  [运单号]
```

### 6.2 数据内容

| 列类型 | 数据来源 |
|--------|----------|
| 客户原始列 | 按 `feedback_export_headers` 映射 |
| 快递公司 | `orders.express_company` |
| 运单号 | `orders.tracking_no` |

---

## 7. 快捷入口

| 入口 | 目标页面 | 说明 |
|------|----------|------|
| 回单导入 | `/return-receipt` | 导入回单 |
| 导出记录 | `/export-records` | 查看导出历史 |
| 订单管理 | `/orders` | 查看订单 |

---

## 8. 注意事项

### 8.1 列名映射
- 只有通过「订单解析」模块导入的订单才有原始列名映射
- 手动创建的订单没有原始列名，会使用系统字段名

### 8.2 导出状态
- 默认仅导出 `returned`（已回单）状态的订单
- 可在高级选项中选择其他状态

### 8.3 按客户打包
- 导出结果按客户打包，每个客户生成一个 Excel 文件
- 多个客户打包为 ZIP 下载
