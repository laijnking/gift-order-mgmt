## [v1.1.1] - 2026-04-20

### 新增功能

#### AI 订单录入
- ✅ **商品选择器**：系统商品编码字段支持从商品档案中选择，可按系统编码或商品名称搜索，选中后自动填充商品名称、规格、品牌等信息
- ✅ **支持未分配供应商提交订单**：订单识别后如未选择发货供应商，允许提交并自动标记为「待派发」状态，方便业务员线下采购后再调整订单状态

#### 订单管理中心
- ✅ **商品选择器**：新增订单和编辑订单对话框中，商品名称字段支持从商品档案中选择系统商品，选中后自动填充系统编码、规格、品牌等信息

## 版本命名规范

```
v{major}.{minor}.{patch}

- major: 主版本号（不兼容的 API 变更）
- minor: 次版本号（向后兼容的功能新增）
- patch: 补丁版本号（向后兼容的问题修复）
```
